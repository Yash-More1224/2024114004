## How to run the code

From the project root, run:

- **Whitebox code:**
  `python whitebox/code/main.py`


## How to run the tests

From the project root, run:

- **Whitebox tests:**
  `python -m pytest whitebox/tests/ -v`

- **Integration tests:**
  `python -m pytest integration/tests/ -v`

- **Blackbox tests:**
  `python -m pytest blackbox/tests/test_quickcart.py -q`
